# /flow-dev — Implement the active feature

1. Verify approved `spec.md` + `plan.md` under `.ai-work/{feature-slug}/`.
2. If `rework_ticket.md` is open → delegate to **forge-dev** for rework first (priority over plan).
3. Delegate to **forge-dev** with plan checklist.
4. Dev: Ralph Wiggum loop (test → fix → repeat until green).
5. Dev must mark completed items `[x]` in `plan.md` (not a human gate; PM-* remain human).
6. Report: implemented scope, tests, checklist progress.

Orchestrator does not edit `src/` or `tests/` inline.
