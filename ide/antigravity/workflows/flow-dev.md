# /flow-dev — Implement the active feature

When invoked, run the Dev phase (Inner Loop):

1. Verify spec.md + plan.md exist and have no blocking questions
2. Delegate to forge-dev with the plan.md task checklist
3. forge-dev implements code + tests using Ralph Wiggum Loop (test → fail → fix → repeat)
4. Report: what was implemented, tests written, checklist items completed
